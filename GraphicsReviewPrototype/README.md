# Graphics Review Prototype

Toto je samostatny prototyp urceny pro rychle overeni vizualniho smeru.

## Co je jeho cil

- overit, jak pusobi scena
- overit siluetu a proporce postavy
- overit, jak vypadaji ruzne polohy
- overit, jak se vizualne projevi obleceni
- overit ruzne lighting presety

## Co to neni

- neni to finalni fotorealisticky render
- neni to `Unity HDRP` build
- neni to gameplay vertical slice

Je to `art direction review build`.

## Spusteni

Otevrit:

- `GraphicsReviewPrototype/index.html`

## Co v nem zkouset

- prepinat polohy postavy
- prepinat svetelne presety
- zapinat a vypinat casti obleceni
- hodnotit, jestli silueta a scena jdou spravnym smerem
