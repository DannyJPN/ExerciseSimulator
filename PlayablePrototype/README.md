# Playable Prototype

Toto je prvni spustitelny lokalni prototyp hlavniho flow.

## Co umi

- spustit `Wall Sit`
- simulovat presun, zarovnani a drzeni pozice
- postupne zvysovat unavu a bolest
- vygenerovat prosbu o ulevu
- nechat trenera prosbu prijmout nebo zamitnout
- umoznit preruseni cviku jinym povelem
- aplikovat sankci `Extend Hold`
- aplikovat sankci `Less Clothing`
- logovat historii udalosti

## Jak to spustit

Otevrit:

- `PlayablePrototype/index.html`

Neni potreba instalace ani build.

## Co je to za typ prototypu

Jde o `UI and logic prototype`, ne o finalni `Unity` build.

Smysl:

- rychle otestovat chovani systemu
- overit trainer UI
- overit tok `wall sit -> relief request -> reject/accept -> failure/sanction`
