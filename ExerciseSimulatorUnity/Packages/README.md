# Packages

Minimalni `manifest.json` je zde jen proto, aby slozka sla otevrit jako projekt.

Po prvnim otevreni v `Unity 6 LTS` doplnit pres `Package Manager`:

- `High Definition RP`
- `Cinemachine`
- `Animation Rigging`
- `AI Navigation`

Pokud zalozite uplne novy `HDRP` projekt v editoru misto otevreni teto slozky, zkopirujte pak do nej aspon:

- `Assets/`
- `ProjectSettings/ProjectVersion.txt`
